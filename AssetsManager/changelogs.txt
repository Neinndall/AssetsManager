AssetsManager - League of Legends | v4.0.0.0

MAJOR UPDATE

New Features
  - Viewer / Mesh Visibility Synchronization
    * Implemented a brand new synchronization system for multiple chromas: when enabled, toggling a mesh part on one model automatically updates its visibility across all other loaded chromas with matching part names. This significantly improves the workflow when analyzing character variations.
    * Integrated a safety-locking mechanism to prevent infinite event recursion during cross-model propagation.
    * Streamlined the collection-changed logic in the 3D scene engine for optimal performance during bulk chroma loading.
  - Settings / Advanced Export Controls
    * Added "Data export target" setting: users can now choose between "Original" (RAW) and "JSON" (Smart) as the preferred format for data extraction. This provides full control over how technical files like `.bin`, `.lua`, and `.stringtable` are handled during extraction processes.
  - Explorer / Standardized Smart Save
    * Enforced Universal Smart Save: the "Save" action now consistently performs high-fidelity conversions regardless of user settings. Images are always saved as `.png` and data files as `.json` (including `.bin`, `.luabin64`, `.preload`, `.troybin`, etc.), ensuring all manual saves are immediately usable and professional.
    * Expanded Data Support: added automated JSON conversion for `.preload` and `.troybin` files during Save and Smart Extract operations.
  - Explorer / Lua Decompilation Support
    * Added full support for `.luabin64` files: the system now automatically decompiles and formats Lua 5.1 bytecode into readable source code.
    * Integrated Automated Smart Save: when using the "Save" command, `.luabin64` files are now automatically decompiled and exported as `.json` data files, streamlining the asset extraction workflow and maintaining consistency with other data formats.
    * Implemented Intelligent Table Reconstruction: advanced logic that groups complex table assignments into clean, readable constructors, matching the original script structure.
    * Integrated Professional Syntax Highlighting: added automated code coloring for `.luabin64` in the explorer previewer, supporting both Lua-specific highlights and structured JSON fallback.
    * Enhanced Opcode Coverage: implemented support for all Lua 5.1 opcodes (TForLoop, Self, Concat, etc.), eliminating "UNKNOWN OPCODE" messages and ensuring perfect script fidelity.
    * Refined Structural Formatting: upgraded the generation engine for superior code layout, featuring cleaned-up conditional logic and human-readable argument naming.
    * Integrated Professional HUD Iconography: assigned the official Lua branding glif to the new file type for a consistent visual experience in the file explorer.
    * High-Performance In-Memory Engine: developed a zero-disk decompiler that processes game data directly in RAM for near-instant visualization.
  - Monitor / Advanced Backup System
    * Implemented Intelligent Installation Discovery: the system now automatically scans for any League of Legends installation by parsing internal metadata, categorizing clients as LIVE or PBE regardless of their folder names.
    * Added Selective Backup Creation: users can now create a backup of any specific installation from the list, providing total control over version cloning.
    * Professional Version-Date Naming: backups are now automatically named using a clean `Folder_Version_Date` format (e.g., `League of Legends PBE_14.9_20260504_1200`), ensuring a perfectly organized historical repository.
    * Dynamic Role Identification: added interactive "MAIN" and "BACKUP" tags to the installation list for immediate visual clarity on which folder is being used.
  - Settings / Explorer Performance Optimization
    * Added "Preferred Directory Folder" setting: users can now choose to load only "Game", "Plugins", or "All" subdirectories when entering Directory Mode in the Explorer. This significantly reduces loading times and RAM consumption for users focused on specific areas of the game data.
  - Audio / Professional Playlist Management
    * Enhanced Playlist Persistence: implemented a dual-URL storage system that preserves the `OriginalUrl` (permanent YouTube link) alongside the temporary stream URI, guaranteeing long-term data integrity for saved packs.
    * Introduced "Playlist Packs": users can now save multiple independent playlists containing both YouTube URLs and local media files.
    * Integrated HUB Interface: added a new header to the Audio Player for quick saving and library navigation.
    * Persistent Storage: playlists are now saved directly to the application configuration, allowing them to be restored instantly upon startup.
  - Text Comparison / Granular Filtering
    * Introduced Granular Filter Controls: added a new "FILTERS" section in the toolbar allowing users to independently toggle the visibility of Insertions (+), Deletions (-), and Modifications (Edit).
    * Implemented Intelligent Filter Activation: clicking any granular filter now automatically activates the "Hide Unchanged Lines" mode, providing an instant focused view of the selected change types.
    * Added "Filter Only" Rapid Isolation (Right-Click): right-clicking on any INSIGHTS badge (Insertions, Deletions, or Modifications) now instantly isolates only that type of change, disabling all other filters for a lightning-fast analysis.
